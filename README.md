System was developed using:

- Ruby version: **ruby 2.3.3p222**
- Ruby on Rails version: **v5.0.1**
- Devise version: **v4.2.0** for authentication
- Thredded version: **v0.9.4** as forums engine
- Sass for styling
- PostgreSQL database
